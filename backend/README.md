# Проект Mesto фронтенд + бэкенд
Ссылка на проект https://github.com/MariaRez/express-mesto-gha.git

## Данная проектная работа включает в себя:
* настройку линтера eslint и  стайлгайда Airbnb;
* создание и подключение к базе данных через сервер MongoDB;
* создание схемы и модели для пользователей;
* создание схемы и модели для карточек;
* создание и использование роутов и контроллеров для пользователей;
* создание и использование роутов и контроллеров для карточек;
* отработку использования сервиса Postman;
* настройка обработки ошибок.


## Директории

`/routes` — папка с файлами роутера  
`/controllers` — папка с файлами контроллеров пользователя и карточки   
`/models` — папка с файлами описания схем пользователя и карточки  
  
Остальные директории вспомогательные, создаются при необходимости разработчиком

## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
